# Contributing

Thanks for helping make agent control more open and interoperable.

1. Open an issue for protocol or trust-boundary changes.
2. Keep pull requests focused and include tests.
3. Run `npm test` in `connector/` and the `AgentKeys` Xcode test scheme.
4. Never commit credentials, provisioning profiles, DerivedData, or captured prompts.
5. Adapter claims must be backed by a reproducible test against the named harness version.

UI changes should preserve Dynamic Type, VoiceOver labels, contrast, and Reduce Motion behavior. Semantic controls must remain usable without relying on color alone.
